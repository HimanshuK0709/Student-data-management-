# Student Data Management System (Python)

This is a beginner-level Python project built to practice core programming concepts.

## Features
- Add student records (roll number & name)
- Store data persistently using JSON
- Load saved data at program startup
- View all student records
- Search student by roll number

## Concepts Used
- Python dictionaries
- File handling
- JSON module (serialization & deserialization)
- Basic error handling

## How to Run
Run the Python file and follow the menu options in the terminal.

## Learning Status
This project was built as part of my learning journey to understand data persistence in Python.
